# Hardware

This directory contains the physical design files for the FootCensor project.

## Mechanical Design
Contains CAD models (e.g., STL, STEP, Fusion 360 files) for:
- Ankle mounting bands
- Sensor array holders (with fixed distance to skin)
- Light-shielding hoods to minimize ambient noise

## Electrical Design
Contains PCB schematics and layouts (e.g., KiCad) for:
- The NIR photodiode array and LED drivers
- IMU integration
- Microcontroller connections (USB interface)
