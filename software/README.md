# Software

This directory contains PC-side scripts and tools for the FootCensor project.
While the final device operates as a standalone USB Gamepad, these tools are required during development for calibration, data logging, and machine learning.

## Components
- **Data Logger**: Synchronized acquisition of raw NIR and IMU values alongside labeled timestamps.
- **Visualizer**: Real-time graphing to evaluate noise, drift, and signal bounds.
- **Model Training**: Pipeline to train the initial classification models (e.g., Logistic Regression, SVM) and evaluate their performance.
